# Indigo Cycling Site

Custom website for [Indigo Cycling Chicago](https://www.indigocycling.cc), 
hosted on GitHub Pages and embedded into Squarespace via iframes.

## Structure
- Each `.html` file corresponds to a page on the site
- `css/styles.css` — global styles
- `js/main.js` — global scripts
- `assets/` — images and fonts

## Pages

|------|----------------|--------|

## Deployment
Hosted via GitHub Pages at:
`https://aprilmariveles.github.io/indigo-cycling-site/`

Changes pushed to `main` go live within ~2 minutes.

## Embedding
Each page is embedded in Squarespace using a Code Block with an iframe.
